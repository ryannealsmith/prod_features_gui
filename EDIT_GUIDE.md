# Edit Functionality Guide

The application already has full edit capabilities for Product Features, Capabilities, and Technical Functions.

## How to Edit Items

### Editing Product Features:

1. Go to the **Product Features** tab
2. Click on a product feature in the left list to select it
3. The details will appear in the right pane
4. Modify any fields you want to change:
   - Label, Name, Platform, ODD, Environment, Trailer
   - When Date, Start Date, TRL dates
   - Details and Comments (text areas)
5. Click **"Save Changes"** button at the bottom
6. A success message will confirm the update

### Editing Capabilities:

1. Go to the **Capabilities** tab
2. Click on a capability in the left list to select it
3. The details will appear in the right pane
4. Modify any fields:
   - Label, Name, Swimlane, Platform
   - Start Date, TRL dates
   - Details (text area)
5. Click **"Save Changes"** button at the bottom
6. A success message will confirm the update

### Editing Technical Functions:

1. Go to the **Technical Functions** tab
2. Click on a technical function in the left list to select it
3. The details will appear in the right pane
4. Modify any fields:
   - Label, Name, Swimlane, Platform
   - Details (text area)
5. Click **"Save Changes"** button at the bottom
6. A success message will confirm the update

## Additional Features

- **Add New**: Creates a new item (opens a dialog)
- **Delete**: Removes the selected item (with confirmation)
- **Filters**: Use platform filters to narrow down the list
- **Relationships**: Manage linked capabilities and technical functions

## Notes

- Fields marked with * are required
- Changes are immediately saved to the database
- The list automatically refreshes after saving
- Date format: YYYY-MM-DD
